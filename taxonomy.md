## ТИПЫ ОБЪЯВЛЕНИЙ (аналог Airbnb)

1. **Снаряжение** — физический предмет напрокат
2. **Опыт / Активность** — гид берёт клиента + снаряжение
3. **Локация** — аренда площадки / базы / водоёма
4. **Пакет** — снаряжение + гид + локация вместе
